## Install
1. `git clone https://github.com/almazkun/django_template.git`
2. `pipenv install`
3. `pipenv shell`
4. `python manage.py startapp <app_name>`
5. `python manage.py makemigrations`
6. `python manage.py migrate`
7. `python manage.py runserver`
